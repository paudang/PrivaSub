# Security Policy

## Supported Versions

We only support the latest version of PrivaSub. Please make sure you are running the latest code or release package.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

Because **PrivaSub** processes local desktop audio feeds, preserving privacy is our absolute priority. 

If you discover a security vulnerability, please **do not open a public issue**. Instead, report the vulnerability directly to the project maintainers:
*   Via email: **security@privasub.local** (or contact the maintainer through their GitHub profile).

We will acknowledge receipt of your vulnerability report within 48 hours and work to provide a patch or mitigation as quickly as possible.
