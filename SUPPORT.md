# Support for PrivaSub

Need help with `PrivaSub`? We are here to help!

## Where to get help

### 1. Documentation
Please check the `README.md` in the repository. We provide detailed information on how to configure and run the application, set up the virtual environment, and use the System Tray menu.

### 2. GitHub Issues
If you encounter a bug, have a feature request, or need help troubleshooting an issue related to the local audio capture or Whisper transcription, please check if an issue already exists before opening a new one.

*   [Search existing issues](https://github.com/paudang/PrivaSub/issues)
*   [Create a new issue](https://github.com/paudang/PrivaSub/issues/new)

When creating a new issue for a bug, please provide as much context as possible:
*   Python version (`python --version`)
*   Operating System (e.g. Windows 11 23H2)
*   Default audio devices (e.g. Speakers, Headphones)
*   The exact error message or unexpected behavior (include console logs if possible)

### 3. Public Roadmap & Task List
We maintain a detailed offline task list and roadmap that you can copy to your personal Trello board:
*   **[View Trello Tickets Task List](trello_tickets.md)**

---

## Security Issues
If you discover a security vulnerability within `PrivaSub`, please do not open a public issue. Instead, report it privately according to our security guidelines in `SECURITY.md`.
